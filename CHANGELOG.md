# Changelog

All notable changes to this project will be documented in this file.

## 0.1.0 - 2026-03-12

- Initial public release of the `readmefy` CLI.
- Repository analysis for Node, Next.js, Vite, Python, Rust, Go, monorepos, and Dockerized services.
- Deterministic README generation with protected marker merge behavior.
- `generate`, `check`, `init`, and `doctor` commands.
- Official GitHub Action and example-driven snapshot coverage.
